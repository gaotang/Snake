## Snake Abstract
  Snake是基于C#开发的实时api监控平台，包括api压力监控和调用异常监控。

  基于RESTFul api风格开发的.net web api微服务应用集群，实时的接口调用监控为系统开发者提供了有利的分析数据。  
  Snake是为此而生的安静而平顺监控系统，它通过Filter实现监控嵌入，异步发送监控日志消息，通过基于RabbitMQ  
  实现的企业服务总线平顺处理消息，最终将监控日志数据输入存储到Mongodb中。

## Requirements
  * Windows 7 +  
  * IIS 7 +  
  * .NET Framework 4.6  
  * RabbitMQ 3.6.5  
  * Mongodb 3.2.1
